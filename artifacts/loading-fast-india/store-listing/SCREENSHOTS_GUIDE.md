# Play Store Screenshots Guide

Play Store के लिए minimum 2 screenshots चाहिए, recommended 4-8.

## Required Specs
- Size: 1080 x 1920 px (portrait) या 1920 x 1080 px (landscape)
- Format: PNG या JPEG
- Max file size: 8 MB each

## Recommended Screenshots (in order)

1. **Home Screen / Trip Booking** — "नज़दीकी ट्रिप तुरंत बुक करें"
2. **Driver Dashboard** — earnings और trip history
3. **Live GPS Tracking** — real-time map view
4. **KYC Verification** — easy document upload
5. **Payment Screen** — transparent pricing
6. **Trip Details** — complete trip info

## Feature Graphic (Required)
- Size: 1024 x 500 px
- यह Play Store listing के top पर दिखता है

## App Icon
- Already configured: `assets/images/icon.png`
- Play Store requires 512 x 512 px high-res icon

## How to take screenshots
1. App को emulator या real device पर run करें
2. हर important screen का screenshot लें
3. Annotate करें (optional but recommended)
4. `store-listing/screenshots/` folder में save करें
